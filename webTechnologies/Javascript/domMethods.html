DOM Manipulation

The DOM represents a web page as a hierarchical, tree-like structure and we can use Javascript methods to manipulate underlying HTML elements of the page in order to create powerful effects.

Nodes

The document tree is a collection of nodes, with each of the branches and leaves on the tree represented by a node. There are a number of different types of node:

Element Nodes
HTML tags e.g. <h2>, <p>, <body> are represented by element nodes within the DOM.
Text Nodes
The text within an element, e.g. the text of a paragraph within the <p> tags are represented as text nodes. In HTML, text nodes must be enclosed within element nodes, but element nodes do not have to contain text e.g. the <img> tag.
Attribute Nodes
Attributes of HTML tags e.g. title or src are represented by Attribute Nodes.

We can use DOM methods to manipulate these nodes.

DOM methods

Finding Nodes

getElementById
Takes argument: an id. Returns: the element with that id. It can only be called on the document object.
getElementsByTagName
Takes argument: the name of a tag. Returns: an array of all the elements with that tag name.
hasChildNodes
Returns true if the element on which it is called has any child nodes.
childNodes
Returns an array of all the children of the element on which it is called.
firstChild
Returns the first child of the element on which it is called.
lastChild
Returns the last child of the element on which it is called.
nextSibling
Returns the next node of the element on which it is called. Or null.
parentNode
Returns the parent node of the element on which it is called. Or null.
previousSibling
Returns the previous node of the element on which it is called. Or null.

These methods are applied within the current scope of the document. So you can find a given element within a document using getElementById, and then use getElementsByTagName to return all the given tags just within that elements scope in the document.

getAttribute
Takes argument: the name of an attribute. Returns: value of that attribute or null if no value is set. It can only be called on an element node.
setAttribute
Takes arguments: an attribute name, the value that you wish to set for that attribute.

Nodes can also be created, inserted, removed and replaced

createElement
Takes argument: the name of the element to be created and returns the new element.
createTextNode
Takes argument: the text to be added. Returns: a new text node containing the specified text. This new text node is not automatically added to the document - you need to use node manipulation methods to insert or append the new node, or replace an existing node.
appendChild
Takes a node as an argument and appends this node as the last child of the element on which it is called. It returns the appended node.
insertBefore
Takes two arguments: the new node and the target node. It inserts the new node before the target node on within the element on which it is called. It returns the new node.
removeChild
It takes a node as an argument and removes this node, and all of its children, from the element on which it is called. It returns a reference to the node that has been removed.
replaceChild
This takes two arguments: the new node and the old node. It replaces the old node with the new node within the element on which it is called. It returns the old node.

You can also access the properties of nodes within the document tree.

nodeName
Returns a string containing the name of the node on which it is called
nodeType
Returns an integer value that corresponds to the type of the node on which it is called.
nodeValue
Returns the value of the node on which it is called.
